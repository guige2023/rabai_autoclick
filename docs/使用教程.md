# RabAI AutoClick 使用教程

## 🎯 快速入门

### 1. 界面介绍

主界面包含以下主要区域：

```
┌─────────────────────────────────────────────────┐
│  工具栏                                       │
```

```
1. 选择动作类型（编辑器左侧）

4. 重复步骤2-3，创建完整工作流
5. 点击"运行"按钮执行
```

## 📝 动作类型详解

### 鼠标点击

**用途**: 在屏幕指定位置执行点击操作

**参数说明**:
- `x`: X 坐标（必填）
- `y`: Y 坐标（必填）
- `button`: 按键类型（left/right/middle，默认left）
- `clicks`: 点击次数（默认1）
- `relative`: 是否相对当前位置（默认false）

**使用示例**:
```json
{
  "type": "click",
  "x": 500,
  "y": 300,
  "button": "left",
  "clicks": 1
}
```

**技巧**:
- 点击参数旁的"选取"按钮，全屏点击选择坐标
- 支持负数延时（相对坐标）

---

### 键盘输入

**用途**: 模拟键盘输入文本

**参数说明**:
- `text`: 输入的文本（必填）
- `interval`: 按键间隔秒数（默认0.05）
- `enter_after`: 输入后按回车（默认false）

**使用示例**:
```json
{
  "type": "type_text",
  "text": "Hello World"
}
```

**变量使用**:
```json
{
  "type": "type_text",
  "text": "用户名: {{username}}"
}
```

---

### 图像识别点击

**用途**: 通过图片模板匹配屏幕上的目标并点击

**参数说明**:
- `template`: 模板图片路径（必填）
- `confidence`: 匹配置信度 0-1（默认0.8）
- `region`: 搜索区域 [x,y,w,h]
- `double_click`: 是否双击（默认false）

**使用示例**:
```json
{
  "type": "click_image",
  "template": "confirm_button.png",
  "confidence": 0.8
}
```

**注意**:
- 模板图片建议使用 PNG 格式
- 置信度 0.7-0.9 适合大多数场景

---

### OCR 文字识别

**用途**: 识别屏幕指定区域的文字

**参数说明**:
- `region`: 识别区域 [x,y,w,h]
- `click_text`: 点击包含此文字的位置
- `contains`: 检查是否包含文字

**使用示例**:
```json
{
  "type": "ocr",
  "region": [0, 0, 800, 200],
  "click_text": "登录"
}
```

**输出数据**:
```json
{
  "text": "识别到的全部文字",
  "results": [
    {
      "text": "单行文字",
      "confidence": 0.95,
      "x": 100,
      "y": 200
    }
  ]
}
```

---

### 延时等待

**用途**: 暂停执行指定时间

**参数说明**:
- `seconds`: 等待秒数（必填）
- `random_deviation`: 随机偏差秒数（默认0）

**使用示例**:
```json
{
  "type": "delay",
  "seconds": 2
}
```

---

### 条件判断

**用途**: 根据条件跳转到不同步骤

**参数说明**:
- `condition`: 条件表达式（必填）
- `true_next`: 条件为真跳转的步骤ID
- `false_next`: 条件为假跳转的步骤ID

**使用示例**:
```json
{
  "type": "condition",
  "condition": "count > 5",
  "true_next": 2,
  "false_next": 3
}
```

---

### 循环控制

**用途**: 控制循环次数和跳转

**参数说明**:
- `loop_id`: 循环标识（必填）
- `count`: 循环次数（必填）
- `loop_start`: 循环起始步骤ID
- `loop_end`: 循环结束步骤ID

**使用示例**:
```json
{
  "type": "loop",
  "loop_id": "main_loop",
  "count": 3,
  "loop_start": 2,
  "loop_end": 7
}
```

---

### 设置变量

**用途**: 设置上下文变量值

**参数说明**:
- `name`: 变量名（必填）
- `value`: 变量值（必填）
- `value_type`: 类型（string/int/float/bool/expression）

**使用示例**:
```json
{
  "type": "set_variable",
  "name": "counter",
  "value": "0",
  "value_type": "int"
}
```

---

### 执行脚本

**用途**: 执行 Python 代码片段

**参数说明**:
- `code`: Python 代码（必填）
- `output_var`: 输出变量名（可选）

**使用示例**:
```python
# 获取变量并计算
a = context.get('a', 0)
b = context.get('b', 0)
return_value = a + b
```

**可用函数**:
- `context.get(name, default)` - 获取变量
- `context.set(name, value)` - 设置变量
- `int(x)`, `float(x)`, `str(x)` - 类型转换
- `len(x)`, `sum(list)`, `min(x, y)` - 内置函数

**安全限制**:
- 不支持 `import` 导入外部模块
- 不支持文件读写操作
- 不支持网络请求
- 不支持系统命令执行

## 🎓 教学模式

### 开启方式

点击工具栏的"🎓 教学"按钮开启教学模式。

### 功能说明

| 功能 | 描述 |
|------|------|
| **按键显示** | 屏幕右上角实时显示按下的键 |
| **组合键显示** | 同时显示多个按下的键 |
| **鼠标点击** | 点击位置显示彩色圆圈 |
| **颜色区分** | 左键绿色、右键红色、中键蓝色 |
| **独立窗口** | 支持最小化，不遮挡主窗口 |

### 特殊键显示

| 按键 | 显示 |
|------|------|
| 方向键 | ↑ ↓ ← → |
| 回车 | ↵ |
| 空格 | ␣ |
| Tab | ⇥ |
| 退格 | ⌫ |
| 删除 | ⌦ |
| Esc | Esc |
| Command (Mac) | ⌘ |

### 使用场景

1. **录屏教学**: 开启教学模式后，执行操作，屏幕上会实时显示按键和鼠标点击，方便录制教学视频
2. **操作演示**: 演示如何使用工具时，观众可以清楚看到每个操作
3. **最小化使用**: 教学模式窗口可以最小化，不影响主窗口使用

## 🔧 脚本编辑器

### 开启方式

点击"🔧 脚本"按钮打开脚本编辑器。

### 功能说明

| 功能 | 描述 |
|------|------|
| **语法高亮** | Python 代码语法着色 |
| **代码模板** | 预设常用代码片段 |
| **测试运行** | 直接在编辑器中测试代码 |
| **变量管理** | 添加测试变量查看结果 |
| **帮助文档** | 内置使用说明 |

### 代码模板

编辑器提供以下预设模板：

1. **基本运算**
```python
# 获取变量并计算
a = context.get('a', 0)
b = context.get('b', 0)
return_value = a + b
```

2. **字符串处理**
```python
# 字符串处理示例
text = context.get('ocr_result', '')
code = text.split(':')[1].strip()
return_value = code
```

3. **条件判断**
```python
# 条件判断示例
value = context.get('result', '')
if 'success' in value:
    return_value = True
else:
    return_value = False
```

4. **循环处理**
```python
# 循环处理示例
items = context.get('items', [])
total = 0
for item in items:
    total += item
return_value = total
```

5. **字典操作**
```python
# 字典操作示例
info = context.get('info', {})
info['processed'] = True
info['timestamp'] = len(str(info))
return_value = info
```

## 💡 实用技巧

### 1. 坐标选取技巧

- 使用"选取"按钮全屏点击选择
- 支持相对坐标（从当前位置偏移）
- 建议先测试坐标是否正确

### 2. 图像匹配技巧

- 模板图片使用清晰截图
- 调整置信度（0.7-0.9 适合大多数场景）
- 使用区域限制提高匹配速度

### 3. OCR 识别技巧

- 缩小识别区域提高准确性
- 确保文字清晰可识别
- 使用 `contains` 参数检查特定文字

### 4. 变量使用技巧

- 使用 `{{变量名}}` 语法引用变量
- 支持表达式计算 `{{a + b}}`
- 变量名建议使用有意义的名称

### 5. 工作流管理技巧

- 为常用工作流添加标签便于分类
- 使用描述性名称便于识别
- 定期清理不需要的历史记录

### 6. 调试技巧

- 使用脚本编辑器测试复杂逻辑
- 查看日志了解执行过程
- 逐步执行验证每个步骤

## ⚠️ 常见问题

### Q: OCR 识别不准确？

**A**: 
- 缩小识别区域
- 确保文字清晰
- 调整置信度

### Q: 图像匹配失败？

**A**: 
- 检查模板图片是否清晰
- 调整置信度
- 确认目标在屏幕可见区域

### Q: 快捷键不生效？

**A**: 
- 确认已安装 `keyboard` 模块
- 检查是否有其他软件占用相同快捷键
- Windows 可能需要管理员权限

### Q: 如何在 macOS 上使用？

**A**: 
- 系统偏好设置 → 安全性与隐私 → 辅助功能
- 添加 Python 或终端到允许列表

### Q: 脚本执行失败？

**A**: 
- 检查语法是否正确
- 确认使用的函数是否在允许列表中
- 查看日志了解详细错误信息
