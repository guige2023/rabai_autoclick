## 功能特性

### 🎯 核心功能

| 功能模块 | 描述 |
|---------|------|
| **可视化编辑器** | PyQt5 拖拽式界面，无需编程即可创建自动化流程 |
| **插件式架构** | 动作模块独立封装，易于扩展新功能 |
| **流程引擎** | 支持顺序执行、条件跳转、循环控制 |
| **变量系统** | 支持变量存储、表达式求值、跨步骤数据传递 |
| **全局热键** | F6 运行 / F7 停止 / F8 暂停（可自定义） |

### 🖱️ 鼠标操作

| 动作类型 | 功能描述 | 参数说明 |
|---------|---------|---------|
| **鼠标点击** | 在指定坐标执行点击操作 | `x`, `y`: 坐标位置<br>`button`: left/right/middle<br>`clicks`: 点击次数<br>`relative`: 是否相对当前位置 |
| **鼠标移动** | 移动鼠标到指定位置（不点击） | `x`, `y`: 目标坐标<br>`duration`: 移动耗时<br>`relative`: 相对移动 |
| **鼠标拖拽** | 从起点拖拽到终点 | `start_x`, `start_y`: 起点<br>`end_x`, `end_y`: 终点<br>`button`: 按键 |
| **鼠标滚轮** | 模拟滚轮滚动 | `clicks`: 滚动格数<br>`direction`: up/down<br>`x`, `y`: 指定位置 |

### ⌨️ 键盘操作

| 动作类型 | 功能描述 | 参数说明 |
|---------|---------|---------|
| **键盘输入** | 模拟键盘输入文本 | `text`: 输入内容<br>`interval`: 按键间隔<br>`enter_after`: 输入后按回车 |
| **按键操作** | 按下特定按键或组合键 | `key`: 单个按键<br>`keys`: 组合键列表<br>`hold_time`: 按住时间 |

### 🔍 图像识别

| 动作类型 | 功能描述 | 参数说明 |
|---------|---------|---------|
| **图像识别点击** | 通过模板匹配定位并点击目标 | `template`: 模板图片路径<br>`confidence`: 匹配置信度(0-1)<br>`region`: 搜索区域<br>`click_offset`: 点击偏移<br>`double_click`: 是否双击 |
| **查找图像** | 查找屏幕上的图像位置 | `template`: 模板路径<br>`confidence`: 置信度<br>`find_all`: 查找所有匹配 |

### 📝 OCR 文字识别

| 动作类型 | 功能描述 | 参数说明 |
|---------|---------|---------|
| **OCR 识别** | 识别屏幕指定区域的文字 | `region`: 识别区域<br>`click_text`: 点击包含指定文字的位置<br>`contains`: 检查是否包含文字<br>`click_index`: 第几个匹配项 |

### 🔧 流程控制

| 动作类型 | 功能描述 | 参数说明 |
|---------|---------|---------|
| **延时等待** | 暂停执行指定时间 | `seconds`: 等待秒数<br>`random_deviation`: 随机偏差 |
| **条件判断** | 根据条件跳转到不同步骤 | `condition`: 条件表达式<br>`true_next`: 条件为真跳转ID<br>`false_next`: 条件为假跳转ID |
| **循环控制** | 控制循环次数和跳转 | `loop_id`: 循环标识<br>`count`: 循环次数<br>`loop_start`: 循环起始ID<br>`loop_end`: 循环结束ID |
| **设置变量** | 设置上下文变量值 | `name`: 变量名<br>`value`: 变量值<br>`value_type`: 类型(string/int/float/bool/expression) |
| **执行脚本** | 执行 Python 代码片段 | `code`: Python 代码<br>`output_var`: 输出变量名 |

### 📷 系统功能

## 详细文档

### 动作类型

#### 🖱️ 鼠标点击 (click)

在指定坐标位置执行鼠标点击操作。

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `x` | int | ✅ | - | X 坐标位置 |
| `y` | int | ✅ | - | Y 坐标位置 |
| `button` | string | ❌ | left | 鼠标按键：left/right/middle |
| `clicks` | int | ❌ | 1 | 点击次数 |
| `interval` | float | ❌ | 0.1 | 多次点击间隔（秒） |
| `move_duration` | float | ❌ | 0.2 | 鼠标移动耗时（秒） |
| `relative` | bool | ❌ | false | 是否相对当前位置 |

**使用示例：**

```json
{
  "id": 1,
  "type": "click",
  "x": 500,
  "y": 300,
  "button": "left",
  "clicks": 1
}
```

**坐标选取技巧：**
- 点击参数旁的"选取"按钮
- 进入全屏选择模式
- 点击目标位置自动填入坐标

---

#### ⌨️ 键盘输入 (type_text)

模拟键盘输入文本内容。

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `text` | string | ✅ | - | 要输入的文本 |
| `interval` | float | ❌ | 0.05 | 按键间隔（秒） |
| `enter_after` | bool | ❌ | false | 输入后按回车 |

**变量替换：**

支持在文本中使用 `{{变量名}}` 占位符，运行时自动替换：

```json
{
  "type": "type_text",
  "text": "验证码: {{captcha}}"
}
```

---

#### 🔍 图像识别点击 (click_image)

通过 OpenCV 模板匹配定位并点击目标图像。

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `template` | string | ✅ | - | 模板图片路径 |
| `confidence` | float | ❌ | 0.8 | 匹配置信度 (0-1) |
| `region` | array | ❌ | null | 搜索区域 [x,y,w,h] |
| `click_offset` | array | ❌ | [0,0] | 点击偏移 [x,y] |
| `double_click` | bool | ❌ | false | 是否双击 |
| `move_duration` | float | ❌ | 0.2 | 移动耗时 |

**使用技巧：**
- 模板图片建议使用 PNG 格式
- 置信度 0.8 适合大多数场景
- 区域限制可提高匹配速度

---

#### 📝 OCR 文字识别 (ocr)

使用 PaddleOCR 识别屏幕文字。

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `region` | array | ❌ | null | 识别区域 [x,y,w,h] |
| `click_text` | string | ❌ | null | 点击包含此文字的位置 |
| `click_index` | int | ❌ | 0 | 第几个匹配项 |
| `contains` | string | ❌ | null | 检查是否包含此文字 |

**输出数据：**

```json
{
  "text": "识别到的全部文字",
  "results": [
    {
      "text": "单行文字",
      "confidence": 0.95,
      "x": 100,
      "y": 200
    }
  ]
}
```

## 开发指南

### 添加新动作

1. 在 `actions/` 目录创建新文件：

```python
# actions/my_action.py
from core.base_action import BaseAction, ActionResult

class MyAction(BaseAction):
    action_type = "my_action"           # 动作类型标识
    display_name = "我的动作"            # 显示名称
    description = "动作描述"             # 详细描述
    
    def execute(self, context, params):
        # 实现具体逻辑
        return ActionResult(
            success=True,
            message="执行成功",
            data={'result': 'value'}
        )
    
    def get_required_params(self):
        return ['param1']               # 必填参数
    
    def get_optional_params(self):
```

## 常见问题

### Q: OCR 识别不准确？

A: 尝试以下方法：
- 缩小识别区域
- 确保文字清晰
- 安装额外的语言包

### Q: 图像匹配失败？

A: 检查以下方面：
- 模板图片是否清晰
- 置信度是否合适（建议 0.7-0.9）
- 目标是否在屏幕可见区域

### Q: 快捷键不生效？

A: 确认：
- 已安装 `keyboard` 模块
- 没有其他软件占用相同快捷键
- 以管理员权限运行（Windows）

### Q: 如何在 macOS 上使用？

A: macOS 需要额外权限：
- 系统偏好设置 → 安全性与隐私 → 辅助功能
- 添加 Python 或终端到允许列表
