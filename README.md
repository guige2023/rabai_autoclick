# RabAI AutoClick v22

桌面端智能自动化工具 - 无代码工作流自动化解决方案

## 产品定位

- **产品名**: RabAI AutoClick
- **类型**: 桌面端自动化工具
- **目标**: 帮助用户自动化重复性工作，提高效率
- **版本**: v22 增强版

## 核心功能

### 工作流编辑器
- 可视化工作流编辑界面
- 拖拽式步骤添加和排序
- 支持循环执行和条件判断
- 工作流保存和加载

### 动作类型

#### 鼠标操作
- **鼠标点击**: 在指定坐标执行单击/双击，支持左键/右键/中键
- **鼠标移动**: 移动鼠标到指定位置
- **鼠标拖拽**: 从起点拖拽到终点
- **鼠标滚轮**: 滚动鼠标滚轮，支持向上/向下

#### 键盘操作
- **键盘按键**: 模拟单个按键
- **组合键**: 模拟组合键如 Ctrl+C、Alt+F4
- **文字输入**: 输入文字内容，支持中英文

#### 图像识别
- **查找图像**: 在屏幕上查找指定图片并返回坐标
- **图像识别点击**: 找到图片后自动点击
- 支持置信度设置
- 支持限定搜索区域

#### OCR 文字识别
- **OCR文字识别**: 识别屏幕指定区域的文字
- 支持点击指定文字
- 支持精确匹配和模糊匹配
- 支持单击/双击、左键/右键选择

#### 系统操作
- **截图**: 截取屏幕指定区域
- **等待**: 等待指定时间
- **执行命令**: 执行系统命令
- **打开应用**: 打开指定应用程序
- **打开网页**: 用浏览器打开网页

#### 流程控制
- **条件判断**: 根据条件跳转到不同步骤
- **循环控制**: 控制循环次数和跳转
- **设置变量**: 设置上下文变量值

### 智能功能

#### 区域选择
- 全屏区域框选
- 选区内透明显示，选区外半透明遮罩
- 实时显示坐标和尺寸信息

#### 坐标选取
- 屏幕点选功能
- 支持所有坐标参数的选取

#### 参数独立存储
- 每个工作流步骤的配置独立保存
- 相同类型动作可以有不同参数

## 安装

### 环境要求
- Python 3.8+
- macOS / Windows / Linux

### 安装依赖

```bash
# 克隆仓库
git clone https://github.com/guige2023/rabai_autoclick.git
cd rabai_autoclick

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### macOS 权限设置

首次运行需要授权辅助功能权限：

1. 打开 **系统偏好设置** → **安全性与隐私** → **隐私** → **辅助功能**
2. 点击左下角锁图标解锁
3. 添加 **终端** 或 **Python** 到列表并勾选
4. 重启程序

## 使用指南

### 基本操作流程

1. **添加步骤**: 在左侧动作列表选择动作类型，点击"添加步骤"
2. **配置参数**: 在右侧配置面板设置参数
3. **设置区域**: 点击顶部"区域"按钮框选识别区域（可选）
4. **执行工作流**: 点击"运行"按钮执行

### 参数说明

| 参数 | 说明 |
|------|------|
| x, y | 屏幕坐标，点击"选取"可在屏幕上点选 |
| region | 识别区域，格式: x,y,宽度,高度 |
| confidence | 图像匹配置信度，0-1之间 |
| button | 鼠标按钮: left/right/middle |
| clicks | 点击次数，双击填2 |
| double_click | 是否双击 |
| condition | 条件表达式，如: ${var} > 10 |
| output_var | 输出变量名，保存结果到变量 |

### 变量使用

支持在参数中使用变量：

1. 使用"设置变量"动作设置变量值
2. 在其他参数中使用 `${变量名}` 引用变量
3. 例如：设置变量 `click_x = 100`，然后在点击动作的 x 坐标填入 `${click_x}`

### 条件判断

条件判断动作支持：

- **condition**: 条件表达式
- **true_next**: 条件为真时跳转的步骤ID
- **false_next**: 条件为假时跳转的步骤ID
