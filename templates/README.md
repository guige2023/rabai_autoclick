# Templates Directory

This directory stores OCR and image match templates for workflows.

## Files

Template images are PNG files used for:
- OCR text recognition templates
- Image match click targets
- Element detection

## Naming Convention

```
YYYYMMDD_HHMMSS_Description.png
```

Example:
```
1Capture_2026-03-06_23.16.00.png
```

## Adding Templates

Templates can be added by:
1. Manual screenshot (PrtScn)
2. Using the built-in screenshot tool
3. Exporting from OCR action

## Best Practices

- Use PNG format for best quality
- Keep templates small (under 100KB)
- Use high contrast images
- Test templates across different screen resolutions

## Cleanup

Remove old templates that are no longer used to save space.
