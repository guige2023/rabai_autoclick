# GUI Directory

This directory contains legacy GUI components (v1).

## Note

This directory contains the original GUI components from earlier versions.
For v22, the main GUI is in the `ui/` directory.

## Files

| File | Description |
|------|-------------|
| `main.py` | Legacy main GUI window |

## Migration

The v1 GUI components have been superseded by the `ui/` package.
New features and updates should be implemented in `ui/`.

## Historical Reference

These files represent the v1/v2 GUI implementation and are kept for:
- Historical reference
- Potential v1 compatibility mode
- Understanding the evolution of the UI

## v1 vs v22 UI Comparison

| Feature | v1 (gui/) | v22 (ui/) |
|---------|-----------|-----------|
| Main Window | Basic | Enhanced with tabs |
| Action Editor | Simple | Advanced with validation |
| Recording | Basic | Process-isolated |
| Diagnostics | None | Full v22 features |
